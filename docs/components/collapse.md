## API

### Collapse API
| 参数        | 说明       | 类型      | 可选值 | 默认值   |
|-----------|----------|---------|-----|-------|
| selected  | 当前激活的面板  | array   | —   | —     |
| accordion | 是否手风琴模式	 | boolean | —   | false |

### Collapse Item API
| 参数    | 说明     | 类型     | 可选值 | 默认值 |
|-------|--------|--------|-----|-----|
| name  | 唯一标志符	 | string | —	  | —	  |
| title | 面板标题	  | string | —   | —	  |
